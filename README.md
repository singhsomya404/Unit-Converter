Project Name : 
Unit Converter Tool Using Python

PROJECT SCOPE :
The Unit Converter Tool is a beginner-friendly Python application designed to perform conversions between different units of measurement. The project aims to simplify mathematical conversions related to length, weight, and temperature through an interactive menu-driven interface.

This project helps users save time and reduce manual calculation errors. It also provides practical exposure to Python programming concepts such as functions, loops, conditional statements, and user input handling.

FEATURES :

Menu-driven interface

Length conversion

Meter ↔ Kilometer

Meter ↔ Centimeter


Weight conversion

Kilogram ↔ Gram


Temperature conversion

Celsius ↔ Fahrenheit

User-friendly input system. 
Continuous execution until exit option is selected. 
Fast and accurate calculations.
